My Skills
Something about my skills

Throught my course of Bachelor of computer applications , I learned alot of programming languages and frameworks which are as follows.

1

HTML

HTML stands for HyperText Markup Language which is the basic requirement for developing websites and web applications.

2

CSS

CSS stands for Cascading Style Sheet which is used for designing the web pages. Without css, it is impossible to attract the users for the website.

3

JavaScript

JavaScript is a scripting language which is used with the primary purpose of DOM manipulaion which means that this language can be used to make real time chnages in the web page as user interacts with it.

4

MERN stack

MERN stack is a combination of various JavaScript frameworks and a database such as MongoDB, Express, React and NodeJs. MongoDB is a database, NodeJs and ExpressJs are together used as backend modules and React is used for frontend development.

5

NextJs

NextJs is a React framework which enhaces the usage of React by adding some new features like app router and pages router which eliminates the complex tasks of express and node.

6

Bootstrap & Tailwind

Bootstrap and Tailwind Css are UI libraries which provides quick solution for designing. It provides readymade classes applied with common CSS which can be directly used in the websites.
